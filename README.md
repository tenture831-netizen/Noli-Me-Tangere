# Noli Me Tangere - Educational Platform

## Pangkalahatang Paglalahad

Ang website na ito ay isang komprehensibong platform na idinisenyo para sa edukasyon tungkol sa klasikong nobela na "Noli Me Tangere" ni José Rizal.

## Mga Bahagi ng Website

### 1. **Header/Title Section**
- Pangunahing titulo at tagline ng website
- Vintage aesthetic background design

### 2. **Kaligiran Pangkasaysayan (Historical Context)**
- Embedded video tungkol sa kaligiran ng Noli Me Tangere
- Deskripsyon ng kasaysayan at kahalagahan

### 3. **Mga Tauhan (Characters)**
- Grid layout ng pangunahing mga tauhan
- Bawat tauhan ay may larawan at paglalahad ng papel

### 4. **Mga Kabanata (Chapters 1-63)**
- Bawat kabanata ay may:
  - Larawan na tumutugma sa kaganapan
  - 1-minute audio recording ng buod
  - Buod ng kabanata

### 5. **Assessment Section**
- **Pre-Test**: Paunang pagsusuri
- **Formative Assessment**: Repleksyon at aktibidad
- **Post-Test**: Panghuling pagsusulit

### 6. **Bokabularyo (Vocabulary)**
- Listahan ng mga salitang makikita sa kuwento
- Kahulugan ng bawat salita

### 7. **Tungkol sa Akda (About the Author)**
- Impormasyon tungkol kay José Rizal
- Timeline ng mga kaganapan

### 8. **Mga Kagamitan para sa Guro (Teacher Resources)**
- PowerPoint presentations
- Lesson plans
- Worksheets
- Discussion guides

## Teknolohiya Ginamit

- **HTML5**: Para sa structure
- **CSS3**: Para sa vintage styling at responsive design
- **JavaScript**: Para sa interaktibong features

## Paano Gamitin

1. Buksan ang `index.html` sa web browser
2. I-navigate ang iba't ibang seksyon gamit ang navigation menu
3. Basahin ang mga kabanata at makinig sa audio recordings
4. Kumpletuhin ang mga assessment activities
5. I-download o i-print ang teacher resources
